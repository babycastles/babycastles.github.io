## setup instructions:

### install homebrew:
`/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

### get ruby going:
`brew install ruby ruby-build`

### install bundler:
`gem install bundler`

### install repo dependencies:
`bundle install`
